# Sky Images

This folder contains paired NIR and RGB images of the sky, captured for analysis in the project. These images are used for the analysis in the report.

## Contents

- **NIR image** – Captured with a near-infrared camera.  
- **RGB image** – Captured with a standard visible light camera.  
- **image_histograms.ipynb** – Jupyter notebook for generating histograms and analyzing image data.

Images are named with timestamps and camera types (`2025-04-15_14-13-00_nir.jpg` and `2025-04-15_14-13-00_rgb.jpg`).
