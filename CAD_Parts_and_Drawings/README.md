# CAD_Parts_and_Drawings

This folder contains all CAD models and technical drawings for the project, including:

- **.SLDPRT** – Individual part models.  
- **.SLDASM** – Assembly model.  
- **.SLDDRW / .PDF** – Technical drawings (plan, elevation, isometric, and see-through views).

Use SolidWorks to view or edit the part (`.SLDPRT`), assembly (`.SLDASM`), and drawing (`.SLDDRW`) files. PDFs are also included which for ease of viewing the drawings
